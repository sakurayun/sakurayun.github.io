# 3.2 剪辑

### 主要工作 <a id="&#x4E3B;&#x8981;&#x5DE5;&#x4F5C;"></a>

根据`单推man` 提供的时间点（[进度公示 - 待剪录播](https://www.notion.so/77e58878ecfd4c489f222244f294cdb2?v=982c8cea43014fb2917ea9a2124ed829)）或是自行寻找直播录像中有趣的内容进行剪辑。

### 注意事项 <a id="&#x6CE8;&#x610F;&#x4E8B;&#x9879;"></a>

#### 关于录播文件 <a id="&#x5173;&#x4E8E;&#x5F55;&#x64AD;&#x6587;&#x4EF6;"></a>

以下内容中所提到录播文件请从`指定获取地址`获取，录播原始文件的获取地址请见**字幕组群公告**。

* 某些录播文件可能为 flv 格式，并无法直接导入 pr/fcpx 进行剪辑，需要转换封装；转换封装应只需要几分钟时间，请不要耗费大量时间重新转码。封装转换工具请见[预处理工具](https://doc.joi-club.cn/#/tool/Pre)。
* 某次录播可能由于网络或推流原因碎掉。如何将碎片文件合成为完整文件，请见[预处理工具](https://doc.joi-club.cn/#/tool/Pre)。

#### 关于视频剪辑 <a id="&#x5173;&#x4E8E;&#x89C6;&#x9891;&#x526A;&#x8F91;"></a>

* 剪辑时请标记区分好自行添加的 BGM 或其他音效；打轴时可能需要无 BGM 版本。
* 剪辑初版时，尽量多选择一些内容，便于在修改意见提出后进行精简。
* 视频制作中请选择使用免费字体，避免使用某些字体。（推荐的字体管理软件：[字由](https://www.hellofont.cn/)）
* 视频中使用各种元素时，请尽量保证一致和美观，可以简单了解平面设计的基础知识。（推荐视频：[零基础平面设计入门系列](https://www.bilibili.com/video/BV1DK4y1b7bY)）
* 剪辑好的视频请通过 `群文件` 进行分享并存档。
