# 3. 工作流程

每个视频大概按以下流程进行制作

> 时间点记录 -&gt; 剪辑 -&gt; 时间轴 / 封面 -&gt; 压制 -&gt; 审核 -&gt; 发布

### 时间点记录 <a id="&#x65F6;&#x95F4;&#x70B9;&#x8BB0;&#x5F55;"></a>

由路灯（单推 man）对直播内容进行标记，着重找出精彩有趣的内容，便于剪辑 man 快速检索视频内容；标记内容整理在[进度公示](https://www.notion.so/77e58878ecfd4c489f222244f294cdb2?v=982c8cea43014fb2917ea9a2124ed829)的`待剪录播`栏目中作为参考，详细内容可以点开栏目内的条目进行查看。

### 剪辑 <a id="&#x526A;&#x8F91;"></a>

由剪辑 man 从`待剪录播`中挑选栏目或是自行挑选感兴趣的直播进行剪辑工作。

### 封面 <a id="&#x5C01;&#x9762;"></a>

美术根据视频内容和标题，进行视频封面的制作。

### 压制 <a id="&#x538B;&#x5236;"></a>

将字幕文件和视频文件压制在一起，最终视频参数应保证清晰度并满足 b 站投稿要求。

### 审核 <a id="&#x5BA1;&#x6838;"></a>

发布前的最后检查工作，确保无误后视频制作工作到此结束。

### 发布 <a id="&#x53D1;&#x5E03;"></a>

由社长/组长将视频和封面上传，发布视频。
